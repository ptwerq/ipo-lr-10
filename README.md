```python

## Задание 4
Создайте функцию `intersectionAreaRect`, которая принимает два аргумента — каждый аргумент представляет прямоугольник в виде списка из двух кортежей `(x, y)`:

1. Первый кортеж — координаты левого нижнего угла прямоугольника  
2. Второй кортеж — координаты верхнего правого угла прямоугольника  

Функция должна возвращать **площадь пересечения двух прямоугольников**.  

- Если прямоугольники не пересекаются — возвращает `0`  
- Если передан некорректный прямоугольник — вызывает `ValueError` с соответствующим комментарием  

**Примеры использования:**

```python
# Прямоугольники пересекаются
intersectionAreaRect([(-3, 1), (9, 10)], [(-7, 0), (13, 12)])  # Вернет положительное число

# Прямоугольники не пересекаются
intersectionAreaRect([(1, 1), (2, 2)], [(3, 0), (13, 1)])      # Вернет 0

# Некорректный прямоугольник
intersectionAreaRect([(1, 1), (2, 2)], [(3, 17), (13, 1)])     # Вызовет ошибку
## Задание 5
Создайте функцию `intersectionAreaMultiRect`, которая принимает **список списков**, где каждый внутренний список представляет прямоугольник в виде двух кортежей `(x, y)`:

1. Первый кортеж — координаты левого нижнего угла прямоугольника  
2. Второй кортеж — координаты верхнего правого угла прямоугольника  

Функция должна возвращать **общую площадь пересечения всех переданных прямоугольников**, учитывая только **уникальные области пересечения без повторений**.  

Если передан некорректный прямоугольник, функция вызывает `RectCorrectError` с соответствующим комментарием.  

Функция должна эффективно обрабатывать любое количество прямоугольников и учитывать все возможные пересечения между ними.

**Примеры использования:**

```python
# Корректные прямоугольники
rectangles = [
    [(-3, 1), (9, 10)],
    [(-7, 0), (13, 12)],
    [(0, 0), (5, 5)],
    [(2, 2), (7, 7)]
]
result = intersectionAreaMultiRect(rectangles)
print(f"Уникальная площадь пересечения: {result}")

# Прямоугольник некорректен
incorrect_rectangles = [
    [(-3, 1), (9, 10)],
    [(3, 17), (13, 1)]  # Некорректный прямоугольник
]
intersectionAreaMultiRect(incorrect_rectangles)  # Вызовет ошибку
